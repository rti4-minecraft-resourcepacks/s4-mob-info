Best Practices

  • Balance mob stats and skills for each tier to ensure fair but challenging encounters.
  • Document all custom skills and stat changes for future reference.
  • Use the recommended player levels as a guideline for dungeon/area design.
  • Test new mobs and skills in isolation before adding to live dungeons.
