Relevant Commands

This page lists useful commands for dungeon development, WorldEdit, and WorldGuard region management.

Dungeon Developer Commands

  • /md edit <dungeon_name> — Enter dungeon editing mode and switch to creative there.
  • /md leave — Leave an active dungeon or exit editing mode.
  • /md play <dungeon_name> — Join a dungeon as a player.
  • /md functioneditor — Gives you a function editor while in editor mode (if you accidentally lose it).

WorldEdit Commands

  • //wand — Gives you the standard WorldEdit selector axe.
  • //___ — All WorldEdit commands use double slashes (//).

WorldGuard Region Commands

  • /rg flag <region> tnt deny
  • /rg flag <region> tnt allow
  • /rg flag <region> tnt
  • /rg flag <region> <flag> <value>
  • /rg define <region-name>
  • /rg redefine <region-name>
  • /rg info [region-name]
  • /rg list
  • /rg remove <region-name>
